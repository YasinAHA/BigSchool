# Calculate Subtotal — Ejemplo de Unit Test (Vitest + TypeScript)

Este proyecto contiene un ejemplo mínimo que implementa `calculateSubtotal` y su test unitario siguiendo el patrón AAA (Arrange–Act–Assert).

Requisitos
- Node.js 18+ instalado

Instalación
En `cmd.exe` ejecutar:

```
cd Testing\Setup_y_MapaDePruebas_FrontBack\01_UnitTest_AAA
npm install
```

Ejecutar tests

```
npm test
```

Estructura relevante
- `src/shared/utils/calculateSubtotal.ts` — implementación
- `src/shared/utils/calculateSubtotal.test.ts` — test (Vitest)
